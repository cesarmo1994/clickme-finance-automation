## Recommended Starting Decision

Use this repository for the MVP with:

- `apps/web` for the frontend
- `apps/api` for the backend
- `docs` for product and technical documentation
- `infra` for deployment scripts once hosting is selected

This keeps the early portal work coordinated while the architecture is still small.
